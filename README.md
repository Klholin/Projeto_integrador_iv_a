# Calculadora Jurídica – Projeto Integrador IV-A
📌 Visão Geral
Este projeto foi desenvolvido como parte da disciplina Projeto Integrador IV-A do curso de Análise e Desenvolvimento de Sistemas da Pontifícia Universidade Católica de Goiás. O objetivo central é aproximar o ambiente acadêmico da realidade profissional, criando uma ferramenta prática para auxiliar escritórios de advocacia e organizações em cálculos jurídicos e financeiros.

🎯 Objetivo
A calculadora jurídica foi criada para otimizar cálculos trabalhistas, cíveis e indenizatórios, reduzindo erros e aumentando a eficiência em processos judiciais. Ela permite realizar operações como:

Horas extras

Férias proporcionais

Rescisão contratual

Décimo terceiro

Juros simples e compostos

Juros de mora

Correção monetária

Indenizações

🛠️ Tecnologias Utilizadas
Java SE 25

JUnit 5 para testes automatizados

Git e Git Flow para versionamento

Maven para gerenciamento de dependências

✅ Validação e Testes
A confiabilidade da calculadora foi assegurada por meio de testes automatizados com JUnit 5. Todos os módulos foram validados com sucesso, incluindo CalculadoraJuridicaTest, CalculadoraTest, CientificaTest, HorasExtrasTest, IntegracaoTest, ParserTest e SuiteCalculadoraJuridicaTest. Os relatórios confirmaram ampla cobertura e ausência de falhas, garantindo precisão nos cálculos de horas extras, férias, rescisão, juros, correção monetária e indenizações. Além disso, o uso prático do menu interativo foi aprovado, demonstrando que a aplicação funciona bem em ambiente de linha de comando, com respostas claras e consistentes às entradas fornecidas.

🌍 Impacto Social
Embora desenvolvido em parceria com o escritório Almeida Campos, o projeto tem potencial de aplicação em outros escritórios de advocacia, ONGs voltadas à defesa de trabalhadores, organizações regionais e microempreendedores. Ele contribui para democratizar o acesso a ferramentas de apoio jurídico, reduzindo erros, otimizando tempo e promovendo maior transparência nos processos. No ambiente acadêmico, fortalece a formação prática dos alunos e aproxima a universidade das demandas reais da sociedade.

🚀 Como Executar
Clone o repositório.

Compile o projeto com Maven (mvn clean install).

Execute os testes com mvn test.

Rode a aplicação via linha de comando para acessar o menu interativo.
